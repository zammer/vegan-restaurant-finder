# Vegan Restaurant Finder

###Getting Started###

Checkout this repo, install dependancies, then start the gulp process with the following:

```
	> git clone git@github.com:zammer/vegan-restaurant-finder.git
	> cd vegan-restaurant-finder
	> npm install
	> npm start
```
